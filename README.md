MiniModules is a pnpm-powered monorepo skeleton with API, admin, mobile, and shared library packages.

# MiniModules

## Getting Started

```sh
pnpm -w install
pnpm -w -r run build
```

## Packages

- `packages/api`
- `packages/admin`
- `packages/mobile`
- `packages/libs`

## Requirements

- Node.js 18+
- pnpm 8+
